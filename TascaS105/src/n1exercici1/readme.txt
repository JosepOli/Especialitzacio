Primer de tot afegim la ubicació correcta de Java a les variables d'entorn.

Un cop creada la classe "N1exercici1" dins del paquet "n1exercici1" la compilem des de la consola amb: javac N1exercici1.java

Per executar aquesta classe, ens ubiquem a la carpeta src i utilitzem el comand: java n1exercici1.N1exercici1 "directori", 
on "directori" és el directori del qual volem llistar els fitxers i directoris.
